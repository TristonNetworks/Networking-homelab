# Lesson 1: Ping, ARP and Traceroute

## Objective
Learn how basic network troubleshooting tools work.

## Tools Used
- Cisco Packet Tracer

## Topics Covered
- Ping
- ARP
- Traceroute

## Network Topology
Two PCs connected through a switch.

## Commands Tested

Ping
ping 192.168.1.2

ARP
arp -a

Traceroute
tracert 192.168.1.2

## What I Learned
- Ping verifies connectivity
- ARP resolves MAC addresses
- Traceroute shows the path between devices
